API ENDPOINT -> BaseURL/Endpoint

Example: https://bored-api.appbrewerycom/random


Query Parameter
bored-api.appbrewerycom/random?query=value  // add a key value pair


bored-api.appbrewerycom/random?query=value&query2=value  // add a multiple query


filter
https://bored-api.appbrewery.com/filter?participants=4



Path Parameter:
https://bored-api.appbrewery.com/Endpoint/{path-parameter} 
example: https://bored-api.appbrewery.com/activity/3943506 // activity: endpoint and a number is a parameter (key)
